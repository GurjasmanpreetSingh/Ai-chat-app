# AI Chat App

An AI-powered chat application built using React.js, Node.js, and the Google Gemini API. The application provides a modern chat experience with markdown rendering, syntax-highlighted code blocks, and chat history management.

## Features

* Real-time AI conversations
* Markdown rendering support
* Syntax highlighting for code blocks
* Chat history management
* Responsive user interface
* Clean and modern chat design

## Tech Stack

### Frontend

* React.js
* JavaScript
* CSS
* React Markdown
* Rehype Highlight

### Backend

* Node.js
* Express.js

### AI Model

* Google Gemini API

## Installation

### Clone the Repository

```bash
git clone <your-repository-url>
cd Ai-chat-app
```

### Install Dependencies

Frontend:

```bash
cd Frontend
npm install
```

Backend:

```bash
cd Backend
npm install
```

### Configure Environment Variables

Create a `.env` file inside the Backend folder and add your Gemini API key.

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Run the Application

Backend:

```bash
npm start
```

Frontend:

```bash
npm run dev
```
